# GeoServer Prompt

Generate GeoServer configuration guidance for a New Zealand renewable energy WebGIS project.

Requirements:

- Workspace name: `renewable_nz`
- PostGIS datastore connected to database `renewable_nz`
- Publish WMS layers for wind suitability, solar suitability, transmission lines, protected areas, and GIR locations
- Use EPSG:2193 as native CRS
- Provide SLD styles
- Include WMS testing checklist
