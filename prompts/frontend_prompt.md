- Responsive layout
- Left-side control panel
- Layer toggle functionality
- Search bar
- Popup information windows
- Legend

- Solar suitability
- Wind suitability
- Transmission lines
- Protected areas
- GIR extracted renewable-energy locations

- Leaflet

- HTML5
- CSS3

The frontend must support:

- Dynamic filtering
- Heatmap rendering
- GeoJSON layers
- WMS overlays
- Popup charts

Generate:

- index.html
- styles.css
- app.js
