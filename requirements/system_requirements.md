# System Requirements

## Functional Requirements

The system shall:

- Display renewable energy suitability layers
- Support WMS integration from GeoServer
- Allow layer switching
- Provide popup information
- Support attribute-based filtering
- Display GIR extracted locations
- Display historical weather resource points
- Support dashboard visualisation

---

## Non-Functional Requirements

The system shall:

- Load within reasonable time
- Be responsive on desktop browsers
- Use open-source technologies only
- Maintain CRS consistency
- Support scalable GIS architecture
